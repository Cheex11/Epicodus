CSS

Here is the code to add css to your html. Add this under your head.

link href="css/styles.css" rel="stylesheet" type="text/css" media="all"

Here is an example of h1 being modified in the css.

h1 {
  color: blue;
  text-align:center;
}

The "h1" is called a selector. "Color" and "text-align" are called properties. "Blue" and "center" are called properties.
You can select individual selectors inside a class using other tags like in the following snipit. The rule is that the most specific selector will always be used. If two rules have the same specificity, the last one wins. Avoid this scenario.

.digital p a {
  background-color: yellow
}
Here is a long list of CSS properties from w3 schools.

Use classes in css to apply styles to sepecific classes. To address a class, first type the selector then "." then the class name." One class can apply to multiple selectors. Use just ."classname" to make changes to all selectors that have the same classname. Use this method to assign a block of text to be grouped together such as the sidebar or footer. ex: div a block to sidebar and use CSS to float the class to the right with a background color of yellow.

"#" is an ID selector which is used to target a single specific element with a unique id. "." is a class selector used to target multiple elements with a particular class. The "#" is used when the style is something you know is only going to appear once, for example things like high level layout divs such as sidebars, banner areas etc.

Use flats to move blocks of code or graphics around the page. Float is a common property.
To make a column, div three sections of code in a "column" class. Use CSS to apply a width and float the class.

Common Operators and syntax:

  • "+","-","*","/" and % which returns the remaider after division.
  • "Comparators": ">", "<", "==" to return boolean value, "!=" NOT equal, ">=", "=<".
  • Use "+" to concatenate strings.
  • Make strings out of expressions to concatenate them.
  • Use "\t" to advance to the next "tab stop".
  • Use a "\"" to show a quitation mark without ending the string too early.
  • Use a double back slash to show a back slash.
  • Use "\n" to shift the printout to a new line.
  • Use ".length" property to return the length of the string.
  • Use "var" to declare a variable. The syntax is "var variable = ##
    • No digits in variable names.
    • Underscores are okay but irritating.
    • CamelCase is the standard
    • Can have numbers
  • To change a variable's value, do not need to re-var.
  • Use "variable += ##" to add to the existing variable. Also works with "*=" and others.
  • To increase the value by one, use VARIABLE++ or VARIABLE-- or "+=###" to subtract.
  • The index of a string will always be one more than the last index. (Because the index starts at zero.)
  • Use "VARIABLE.charAt(##)" to find the character at a specific index.
